
> # to-vim

![vim](./images/vim.png)

### 安装
```
sudo apt-get install vim  // Ubuntu
```

### 新手指南
```
vimtutor  // vim 教程
```

### 配置
```
cd Home               // 进入 Home 目录
touch .vimrc          // 配置文件
touch .vimrc.bundles  // 插件管理文件
```

###  使用 vunble 管理插件
```
$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
